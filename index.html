booster-data

explicit card pool definitions for mtg booster products

about

this project provides ground-truth data for which cards can appear in each mtg booster pack type. no inference, no general rules - just explicit collector number ranges per slot.

usage

// load the manifest
const index = await fetch('https://bensonperry.com/booster-data/index.json').then(r => r.json());

// load a specific booster file
const blbPlay = await fetch('https://bensonperry.com/booster-data/boosters/blb-play.json').then(r => r.json());

file format

each booster file defines slots with collector number ranges:

{
  "set": "blb",
  "setName": "Bloomburrow",
  "boosterType": "play",
  "source": "https://magic.wizards.com/...",
  "slots": [
    {
      "name": "rare",
      "count": 1,
      "rarities": ["rare", "mythic"],
      "pool": { "nonfoil": ["1-261"] },
      "mythicRate": 0.125
    }
  ]
}
